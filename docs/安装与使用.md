# 安装与使用

## 方式一：直接复制提示词

打开：

`prompts/寓言故事一稿通公开版提示词.md`

复制全文到一个新的 AI 对话中，然后按提示补充主题。如果不填主题，工作流会自动从公开热门内容中提炼候选词条，并自行选择一个低风险题材继续执行。

## 方式二：作为 Codex Skill 使用

把目录：

`skills/fable-one-draft-public`

复制到本机 Codex Skill 目录：

`.codex/skills/fable-one-draft-public`

之后在 Codex 对话中描述类似需求：

```text
请用寓言故事一稿通公开版工作流，自动选题并生成小红书漫画分镜和知乎寓言成稿。
```

## 建议输入

```text
请按寓言故事一稿通公开版完整流程执行。
主题为空时请自动选题，不要等待我确认。
输出小红书 9 页漫画分镜、图片 worker 提示词、小红书发布文案、知乎寓言成稿和 QA 清单。
不要发布平台，不读取账号凭据。
```

## 注意

公开版只提供工作流，不保证任何平台发布效果。涉及医疗、金融、法律、未成年人、灾难、刑事、隐私、政治等高风险内容时，应跳过或只做机制层面的温和讨论。
